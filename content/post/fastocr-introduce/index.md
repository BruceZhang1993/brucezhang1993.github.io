小巧易用的桌面 OCR 文字识别工具，原生 QML GUI 应用界面，支持多个 OCR 服务提供者，支持高精度模式和多语种（部分提供者），支持 DBus 外部调用，提供简易的图形设置界面。

    Recommanded on Linux, supported on Windows/macOS.

![Screenshot](screenshot.png)


目前支持的 OCR 提供者

- 百度 AI 文字识别服务 https://ai.baidu.com/tech/ocr
- 有道文字识别服务 https://ai.youdao.com/product-ocr-print.s
- 旷视 Face++ 文字识别服务 (Experimental) https://www.faceplusplus.com.cn/general-text-recognition/


特性

- 通用文字识别
- 多语言支持 / Baidu
- 高精度接口支持 / Baidu
- DBus 支持 / Linux


支持的平台

- Linux(X11)
- Linux Wayland (KDE/Gnome/Sway...)
- Windows
- macOS
